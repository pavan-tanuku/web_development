# Todo List App

A small beginner-friendly Todo List application that allows users to add tasks and mark them as completed.

## Features
- Dynamic DOM manipulation
- Mark tasks as completed or not completed
- Display completed tasks clearly

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
- Manipulating the DOM dynamically using JavaScript
- Handling user interactions
- Managing simple application state

## Future Improvements
- Store tasks using a database or local storage
- Add animations for better user experience
